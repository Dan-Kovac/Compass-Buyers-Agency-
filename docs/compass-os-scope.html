02 · The off-rampStep off at any milestone boundary.

From the Statement of Work
"Compass can pull out at any milestone boundary with full ownership of everything delivered to that point. No further fees. No clawback."

Each milestone earns the next. Sign-off at every boundary. If a milestone isn't accepted, the next invoice doesn't issue and the work continues until it is. The strongest commercial off-ramp is at M2 sign-off, $39,000 in: you've got the audit foundation plus the Sales Engine in the team's hands. M3 and M4 only commit Compass to the final $11,000 once you've seen M2 land.

04 · Beyond Stage 1What's possible once the foundation is in.

Franchise & scale
Multi-tenant activation
Architecture is multi-tenant from M1. Activating a second tenant (franchise, partnership, licensee) is a switch flip, not a rebuild.
Phase 2
Semantic matching
Beyond rule-based. The matcher learns from feedback and surfaces properties on intent, not just attributes. Built on pgvector.
Phase 2
External property enrichment
CoreLogic and PriceFinder data flowing into the log. Valuation history, ownership, comparable sales.
Phase 2
Historical structured parsing
Stage 1 ingests history as archive-grade. Phase 2 turns every past WhatsApp thread and email into structured data.
Phase 2
Daily briefings
Auto-generated team standup from the night's activity. Needs more than four weeks of data to be useful.
Phase 2
Conversion optimisation agent
A/B tests the sales motion. Learns which proposal angles convert, which framings close.
Phase 2
Bruce in client groups
Stage 1 keeps Bruce as silent observer when added to client groups. Phase 2 lets Bruce participate with clients directly.
Phase 3
Suburb reports, productised
Stage 1 has the skill version in the team's hands. Phase 3 turns it into a client-facing product Compass can publish, sell, or use as a lead magnet.
Phase 3
White-label Compass OS
License the system to other buyers agencies. Compass becomes the operating system for the category.
Phase 3
Voice-first interface
Call Bruce, don't just message. Driving between inspections, dictating a brief, asking for the next three matches — all voice.
