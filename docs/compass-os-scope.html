Delivery

Project milestones

Each milestone is tangible enough to stand alone. If the project ended at any sign-off gate, Compass keeps a complete, working system that's a success on its own. More successful with the full build.
M1
Weeks 1-4
Compliance + Foundation
What's live at sign-off
  • Secure multi-tenant OS app, login for every team member
  • Per-client compliance folders (REBAA, QLD + NSW Fair Trading)
  • Audit log on every action — who did what, when, on which client
  • One-click compliance export per client (v1 zip)
  • Every past Compass client filed in audit-ready structure
  • Bruce live in team WhatsApp DMs: voice notes, photos, forwards auto-filed
  • Dedicated bruce@compassbuyersagency.com.au inbox
  • Compass Anthropic workspace stood up
  • Brief Extractor + Compliance Filer agents live
  • First skills in the team's hands by week 3
  • Dashboard v0: active clients, files captured, archive coverage
If the project ended here Compass becomes audit-defensible from day one. Every past client filed. The licensee can sleep. PI insurers see a tighter operation.
M2
Weeks 5-8
The Sales Engine
+ All M1 deliverables remain in production
What's added at sign-off
  • Sales pipeline view: every deal across the 4-phase flow, real-time
  • Client profile with proposal history + version log per client
  • Live proposal URL: one canonical link per client, refreshes on edit
  • Bruce drafts proposals in WhatsApp from voice notes or chat
  • Iterate proposals in DM ("redo the strategy section") — Bruce regenerates
  • Voice notes transcribed and filed automatically
  • Proposal Writer + Proposal Editor agents live
  • Match-making skill in the team's hands
  • Compliance summary skill in the team's hands
  • Dashboard v1: sales velocity, turnaround time, out vs signed
If the project ended here Discovery to live proposal in hours, not days. Same-day proposals. Conversion lifts. Sales velocity is a number on the dashboard, not a feeling.
M3
Weeks 9-12
Property Log + The Close
+ All M1, M2 deliverables remain in production
What's added at sign-off
  • Property log browser: search, filter, every property the team's heard of
  • Manual entry for off-market properties (photos, notes, source)
  • Property Parser live — drop a RE.com.au or Domain link, Bruce structures it
  • Property Matcher (rule-based v1): new brief scans log, new property scans briefs
  • Real-time match alerts to the team when there's a hit
  • Bruce captures close-phase outcomes (offers, negotiations, decisions)
  • Forwarded emails ingested via the Bruce inbox
  • Per-client compliance file UI in the app
  • CAF auto-population from brief → DocuSign → signed → engaged, zero manual handoff
  • Website editor + onboarding kit drafter skills in the team's hands
  • Client portal Phase 1 live (read-only): brief, matched properties, document vault, key dates
  • Dashboard v2: pipeline funnel, time-in-stage, match rate, intake-to-paid
If the project ended here The property log compounds. Matches surface in real time. Closed-loop onboarding eliminates manual handoffs. Compass's biggest piece of IP is captured systematically.
