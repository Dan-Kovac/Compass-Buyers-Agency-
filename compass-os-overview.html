Client security, by design

Buyers agency clients hand over financial details, family circumstances, and large sums of money. The system has to treat that the way a private bank would.

  • AU data residency. Every byte of client data sits in a Sydney AU-region data centre. Nothing leaves the country.
  • Compass owns every account. Cloud, database, domain, secrets, API keys. All in Compass's name. Roadmap holds nothing.
  • SSO + 2FA mandatory on every admin account. No shared passwords, no key person dependency.
  • Secrets in a managed vault (1Password Business or AWS Secrets Manager). No keys in code, no keys in chat.
  • Encrypted at rest and in transit across the whole stack.
  • Daily automated backups with point-in-time recovery on the database.
  • Audit log inside the OS itself. Who exported what, when. Defensible if a client or regulator asks.
  • Annual security review baked into the maintenance plan, not a one-off.

The Claude workspace

One Anthropic organisation, in Compass's name, billed by Compass. Recommended plan: Claude Enterprise. The reasoning is straight security maths.

  • Zero data retention on API traffic by default. Client information passes through, isn't stored, isn't used for training.
  • SSO across the team. One identity, removable in one click when someone leaves.
  • Full audit log of who used what, when. Same compliance posture as the OS itself.
  • Single billing relationship. Skills, Bruce, agents, and Claude Code seats all run on the same org. One place to see usage, one invoice.

If a regulator asks "where does client data go when your team uses AI?", the answer is one paragraph: Compass-owned Anthropic Enterprise org, AU traffic, zero retention, full audit trail.

On the Mac Mini we discussed

Worth flagging directly because it came up in our conversations. The argument for a Mac Mini at the office was tangible local ownership: hardware Compass can see and touch. Real, but it solves the wrong problem.

A box at the office is a single point of failure. One power cut, one cleaner unplugging the wrong cable, one office break-in, and client data is offline or worse. It also doesn't scale to franchise #2 without buying another box and another set of headaches. Sydney AU-region cloud satisfies data residency, has 99.99% uptime guarantees, and is in Compass's name the same way a Mac Mini would be.

The Mac Mini stays useful as an optional Claude Code workstation at the office for the team to work on the OS. It's never the production host.
